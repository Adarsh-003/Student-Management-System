## Assignment Submission
Student Management System

-- Project Description

This is a Java console-based Student Management System developed using Object-Oriented Programming (OOP) principles. The application allows users to manage students, courses, and enrollments through a menu-driven interface.

Features
-Add, view, search, remove and deactivate students
-Add, view and deactivate courses
-Enroll students into courses and viewEnrollments 
-Update enrollment status (ACTIVE, COMPLETED, CANCELLED)
-Automatic ID generation using a utility class
-Basic exception handling with custom exceptions

How to Compile
-javac src/com/airtribe/ui/Main.java


How to Run
-java src/com/airtribe/ui/Main
