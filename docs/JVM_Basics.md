JVM Basics

What is JDK?

JDK is a software package used to develop Java applications. It contains tools such as the Java compiler, debugger, and the JRE. 



What is JRE?

JRE provides the environment required to run Java applications. It contains the JVM and the necessary libraries.



What is JVM?

JVM is the component responsible for running Java programs. It executes Java bytecode and converts it into machine-specific instructions. The JVM also manages memory and garbage collection.



What is Bytecode?

When a (.java file) is compiled using the Java compiler, it is converted into a .class file. This .class file contains bytecode.

Bytecode can run on any system that has a JVM installed.


What does "Write Once, Run Anywhere" mean?

Java follows the principle of "Write Once, Run Anywhere" (WORA). A Java program is compiled into platform-independent bytecode instead of machine-specific code.

The same .class file can run on different operating systems such as Windows, Linux, and macOS without changing the source code.
